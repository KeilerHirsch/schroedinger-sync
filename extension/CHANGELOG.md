# Changelog

## 1.5.0 (2026-03-09)

- Initial VS Code Extension release
- Statusbar sync button with live count
- Auto-Sync with FileSystemWatcher (debounced 5s)
- JSONL parser (TypeScript port of Python v1.0)
- Markdown summary generator
- State tracking (skip unchanged sessions)
- Git auto-commit (optional)
- 3 configurable settings: autoSync, outputDir, gitAutoCommit

## 1.0.0 (2026-02-28)

- Python CLI release
- JSONL session parsing
- Markdown summary generation
- Git auto-commit support
- State tracking with hash-based change detection
